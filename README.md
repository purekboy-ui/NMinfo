# 義大核醫排檢筆記 (NMinfo)

這是我平常工作用的備忘錄，用來快速查詢核醫檢查的規範、排程禁忌，還有些這幾年整理下來的病人衛教話術。

這不是什麼大系統，單純是一個方便自己和同事查找資料的靜態網頁。

👉 **線上版**：[GitHub Pages](https://purekboy-ui.github.io/NMinfo/) | [Firebase](https://nminfo-f108e.web.app/)

## 內容物

*   **檢查規範查表**：收錄 Bone Scan, MPI, PET/CT 等常見項目的適應症、藥物劑量、掃描時間。
*   **衛教防呆**：病人常問的「有沒有輻射？」、「要不要停藥？」這類問題，這邊有整理標準回答。
*   **行政備忘**：像是一些檢查的硬性規定 (Hard Rules)，比如 I-131 住院排程限制、急救車盤點流程等。
*   **無聊做的東西**：順手整合了之前寫的 [PET 智慧排程助手](https://nminfo-petarr.web.app/) 和其他排程 DEMO 連結，方便切換。

## 技術與設計

因為只是靜態筆記，不想搞太複雜，全站只用最基本的 HTML / CSS / JS 手刻 (Vanilla JS)，沒有用任何前端框架。

*   **視覺風格**：雖然是內部工具，但介面還是盡量弄得好看一點，用了點 Glassmorphism (毛玻璃) 效果，看著心情比較好。
*   **手機版也通**：有特別調過 CSS，手機查資料或是用平板看應該都沒問題。
*   **資料維護**：所有資料都直接寫在 JS 裡，如果要修文字或新增檢查項目，直接改 code 比較快。
