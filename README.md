# 義大核醫排檢筆記 (NMinfo)

這是一個用於快速查詢核醫檢查規範、排程禁忌，以及病人衛教資訊的靜態網頁系統。

👉 **線上版**：[NMinfo Web App](https://nminfo-f108e.web.app/) 或 [GitHub Pages](https://purekboy-ui.github.io/NMinfo/)

## 內容物

*   **檢查規範查表**：收錄常見項目的適應症、藥物劑量、掃描時間。
*   **衛教防呆**：整理標準化病人衛教回答。
*   **行政備忘**：包含各項檢查硬性規定與流暢作業須知。
*   **工具箱**：整合各式相關排程輔助網頁連結。

## 技術與設計

本專案為純靜態網頁 (Vanilla HTML / CSS / JS)，無需依賴前端框架。視覺採用 Morandi 色系的 Glassmorphism (毛玻璃) 風格，並支援跨裝置響應式 (RWD) 瀏覽。所有資料均封裝於 JavaScript 內，易於維護與修改。
